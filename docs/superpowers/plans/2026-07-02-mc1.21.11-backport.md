# MC 1.21.11 Backport Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the full v0.5.0 feature set (Paper + Folia support, protocol v16, soak harness) build, test, and release for Minecraft 1.21.11 on the long-lived `support/mc1.21.11` branch.

**Architecture:** This is a *version retarget*, not a rewrite: research (committed at `docs/superpowers/specs/2026-07-02-mc1.21.11-backport-research.md`) compile-verified that `paper/src/main` needs **zero** source changes against the 1.21.11 dev bundle, and the entire Fabric+common source delta is three mechanical rename groups plus two Java-22 unnamed variables. The work is concentrated in build files, metadata, regenerated golden fixtures, the soak harness's world-layout assumption, and the release workflow's dual-line versioning.

**Tech Stack:** Fabric Loom (`fabric-loom-remap`), fabric-api 0.141.4+1.21.11, paperweight-userdev 2.0.0-beta.21 + dev bundle `1.21.11-R0.1-SNAPSHOT`, Java 21 bytecode, Paper build 132 / Folia build 14 (both STABLE on fill.papermc.io v3).

## Global Constraints

- **Never touch `main`** — every commit in this plan lands on `support/mc1.21.11` only. Never reuse a bare `vX.Y.Z` tag; support tags are `vX.Y.Z+mc1.21.11`.
- **Java 21 bytecode everywhere**: `options.release = 21` in fabric and paper (common already 21); mixin `compatibilityLevel: JAVA_21`. MC 1.21.11 clients/servers run JRE 21 — class-file 69 will not load.
- **Protocol v16 unchanged** — no wire-format edits. Golden NBT fixtures WILL differ from main's (registry ids are version-baked); regenerate on this branch, never copy fixtures between branches.
- **Folia support stays** (Folia 1.21.11 build 14 is SUPPORTED/STABLE) and stays labeled **experimental**.
- Version pins (research-verified 2026-07-02): `minecraft_version=1.21.11`, `loader_version=0.19.3` (keep), `fabric_version=0.141.4+1.21.11`, loom plugin `net.fabricmc.fabric-loom-remap` (obfuscated-MC variant; 26.x-only `net.fabricmc.fabric-loom` has no mappings step), paperweight-userdev `2.0.0-beta.21` (keep — compile-verified against the old bundle), sodium `mc1.21.11-0.8.12-fabric`, modmenu `17.0.0`.
- Loom-remap version: try `1.15.4` (newest released 1.15.x) on the current Gradle 9.5.1 wrapper first; fallbacks in Task 1 Step 3.
- Reference for "how did this look on 1.21.11": `git show v0.2.3:<path>` (last 1.21.11 release). v0.2.3 predates the soak harness, Paper Tier-1 tests, and Folia — those exist only at HEAD.
